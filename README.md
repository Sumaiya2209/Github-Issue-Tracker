<h3>1️⃣ Difference between var, let, and const?</h3>

<ul>
  <li><strong>var</strong> can be redeclared and updated.</li>
  <li><strong>let</strong> can be updated but not redeclared in the same scope.</li>
  <li><strong>const</strong> cannot be reassigned after declaration.</li>
</ul>

<h3>2️⃣ What is the spread operator (...)?</h3>
<p>The spread operator (...) expands elements of an array or object into individual values.
It is often used to copy arrays/objects or merge them easily.</p>

<h3>3️⃣ Difference between map(), filter(), and forEach()?</h3>

<ul>
  <li><strong>map()</strong> creates a new array by transforming each element.</li>
  <li><strong>filter()</strong> creates a new array with elements that match a condition.</li>
  <li><strong>forEach()</strong> simply loops through the array and performs an action.</li>
</ul>

<h3>4️⃣ What is an arrow function?</h3>
<p>An arrow function is a shorter syntax for writing functions using =>.
It makes code cleaner and is commonly used in modern JavaScript.
</p>

<h3>5️⃣ What are template literals?</h3>
<p>Template literals are strings written with backticks ( ) instead of quotes.
They allow inserting variables directly inside strings using ${variable}.</p>
